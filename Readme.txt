線上問答遊戲:
[動機]
遠距上課期間，同學常因注意力不集中而降低學習成效，導致課堂參與率降低。為了解決此問題，此系統引入搶答與加分機制，鼓勵同學積極參與回答問題，讓同學在輕鬆有趣中學習，並提升課堂互動率。
[系統介紹]
老師端 : 輸入題目及答案以即時發布題目，結束遊戲後，系統會跳出本次遊戲參與者的分數排行榜，可作為老師加分的依據。
學生端 : 爭取答題權，回答正確可得分，回答錯誤則釋放答題權，重新開放搶答。

home.html 為一開始的主頁面
gameclient.html 為答題者
gamehost.html 為出題者(即主持人)
serverH.py 要先開啟才可玩遊戲
(pip install websockets --user)
成果展示資料夾為各操作畫面的截圖

裡面的防呆:
1.只有一位主持人可以連線
2.玩家不可取一樣的暱稱
3.當沒有玩家連線時，主持人不可開啟遊戲
4.欄位有空白不可提交訊息
5.玩家不可取有#跟*跟,的字元
6.遊戲開啟後其他玩家不可進入
